  • The data frame is downloaded from [1] and includes data on mental health of medical students.
  • The data frame has 886 rows and 20 columns.

  • There are 0 missing data.
  • Also among the interval data, outliers are detected and removed. However, there were very few outliers.

Part 2: Planning

As an initial evalutation of association between data, the correlation matrix of interval data are calculated using Pearsons method. The result suggests a good correlation between MBI-EX (a measure of emotional exhaustion) and CESD (a measure of depression) with a correlation coefficient of 0.585, so they are selected for the rest of the study.

  • qcae_aff: affective empathy score
  • mbi_ex: MBI emotional exhaustion

Statistical summary of each data is reviewed. They are also plotted to visually explore their relationship. It is worth mentioning that both variables are assumed interval variables.
A correlation hypothesis test can be performed to provide statistical statement with statistical significance. This, however, requires test of normality. The qq-normal for both variables exhibit satisfactory normality. Histograms and QQ-nomral plot of both variables suggest that CESD is non-normal and is very positive-skewed, but MBI-EX seems almost normal. To ensure this, the result of normality test, Skewness.2SE and Kurtosis.2SE are observed. The results supports the conclusion from the histogram plot. However, normality test rejects the normality of both variables.

  • Despite the qq-normal plot, based on Shapiro-Wilk test, both MBI-EX and CESD are non-normal with \(p=\) 2.02e-05 and \(p=\) 1.03e-13 respectively i.e.\(p*<0.05\). Therefore, correlation test cannot be performed. Although the sample size is 810 (more than 30) because of highly skewed distribution of CESD, transformation is tried to make the data normal thus reducing error as much as possible.

  • The transformation can be applied only on CESD, but then the correlation coefficient would not show a linear relationship, which is the target of this study

  • The transformation is to improve normality of CESD, however, because MBI-EX data has already very little skewness, the transformation may increase its skewness and reduce its normality. So, this trade-off needs to be considered in selection of the transformation function.

  • The applied transformation is \(X^{0.7}\). It is considered that the range of variables is positive, so the transformation can be applied for the whole range.

  • The transformation has eliminated most of the skewness, but there is still some kurtosis for CESD. However, considering the satisfactory qq-normal and the good sample size (>30), the Pearsons correlation test is performed on the variable pair i.e.both assumptions of this test, which are interval data and normality (or sample size>30), are satisfied.

Part 3: Analysis

  • The correlation coefficient is 0.601 with the 95% confidence interval of (0.556, 0.644).

  • Because p-value is 7.63e-81 (\(*p<0.05\)), the null hypothesis is rejected i.e.there is correlation between MBI emotional exhaustion and affective empathy at 5% level of significance.

  • Instead of transforming data, we can change our method to Spearman, which does not required normality and interval data. The p-value of Spearmans method would be 4.16e-81, which also rejects Null hypothesis i.e.supports correlation.

  • The confidence interval of Spearmans can be obtained from bootstrapping and be compared with the one from Pearsons.

  • The confidence interval from basic bootstrapping of Spearmans method on non-transformed data is 0.557, 0.652. This CI is very close to the one from Pearsons method: 0.556, 0.644.

detect_outlier <- function(x) {
  
  Quantile1 <- quantile(x, probs=.25, na.rm = TRUE)
  Quantile3 <- quantile(x, probs=.75, na.rm = TRUE)
  IQR = Quantile3-Quantile1
  x > Quantile3 + (IQR*1.5) | x < Quantile1 - (IQR*1.5)
}

# Takes a data frame and a column and removes the outliers from the
# original data frame
remove_outlier <- function(dataframe,
                           columns=names(dataframe)) {
  # for loop to traverse in columns vector
  for (colmn in columns) {
    # remove observation if it satisfies outlier function
    dataframe <- dataframe[!detect_outlier(dataframe[[colmn]]), ]
  }
  return(dataframe)
}
